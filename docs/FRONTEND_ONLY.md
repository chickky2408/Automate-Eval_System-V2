# Frontend Only — ขอบเขตงาน

**หน้าที่ของ repo นี้ (สำหรับฉัน): ทำเฉพาะ Frontend เท่านั้น**

- ✅ แก้/เพิ่มเฉพาะไฟล์ในโฟลเดอร์ **`frontend/`**
- ❌ **ห้ามแก้ backend เด็ดขาด** — ไม่ใช่หน้าที่ของฉัน
  - ห้ามแก้: `backend/` ทุกไฟล์ (routers, services, db, main.py, uploads, ฯลฯ)
  - ห้าม commit ไฟล์ backend

เวลา commit ให้ add เฉพาะไฟล์ภายใต้ `frontend/` (และถ้าต้องการ root `package.json` ที่ใช้รัน frontend จาก root)
