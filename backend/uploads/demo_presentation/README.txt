Demo files for UX/UI presentation
================================

.erom = ERoM (firmware/binary) — use with "Select ERoM" in Test Setup
.ulp  = ULP (optional LIN) — use with "Select ULP (optional)" in Test Setup

You can drag & drop these into the app's file upload area during the demo.

Files:
- demo_erom_1.erom, demo_erom_2.erom, demo_erom_3.erom
- demo_ulp_1.ulp, demo_ulp_2.ulp
